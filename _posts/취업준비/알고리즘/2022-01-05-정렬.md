## 버블정렬
+ 인접한 두 원소를 비교하는 정렬
+ 시간복잡도 : O(N<sup>2</sup>) 이미 정렬된 경우 O(N)

'''java
// boolean swapped = false;
for (int i=1; i<size; i++) {
    for (int j=0; j<size-i; j++) {
        if (a[j] > a[j+1]) {
            swap(a, j, j+1);
            // swapped = true;
        }
    }
    // if (swapped == false) {
        break;
    }
}
```
